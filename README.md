# Habit Streak Tracker

A beautiful, responsive web application for tracking daily habits and maintaining streaks. Built with React, TypeScript, and Tailwind CSS.

## Features

- Add/delete habits with name and emoji
- One-tap daily habit check-off with smooth animations
- Streak tracking with milestone celebrations (confetti at 7, 30, 100, 365 days)
- Calendar history view with monthly statistics
- Data persists in localStorage (no account required)
- Automatic midnight reset
- Responsive design (mobile, tablet, desktop)
- PWA support (installable, works offline)
- Export data as JSON

## Tech Stack

- **React 18+** with TypeScript
- **Tailwind CSS v4** for styling
- **Vite** for build tooling
- **canvas-confetti** for milestone celebrations
- **localStorage** for data persistence

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build outputs to `dist/`.
